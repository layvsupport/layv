프로젝트를 실행하려면 다음 단계를 순서대로 따라 하세요.

1. Node.js와 npm이 설치되어 있는지 확인하세요.
2. Windows에서 PowerShell이나 명령 프롬프트를 열고 현재 위치가 `C:\Users\parkc>`처럼 보인다면, 다음 명령으로 프로젝트 루트(`AISpace`)로 이동하세요.
   ```
   cd /d D:\BlendBytes\AISpace
   ```
   이동이 완료되면 프롬프트가 `D:\BlendBytes\AISpace>`처럼 바뀐 것을 확인할 수 있습니다.
3. 프로젝트 루트에서 `npm install`을 실행해 필요한 패키지를 설치하세요.
4. 설치가 완료되면 `npm start` 명령으로 프로그램을 실행하세요.

추가 팁:
- 패키지를 다시 설치해야 할 경우 `npm install`을 다시 실행하면 됩니다.
- 실행 중 문제가 발생하면 터미널 메시지를 확인해 오류 내용을 파악하세요.
