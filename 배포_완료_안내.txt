=== ✅ AISpace 배포 파일 생성 완료! ===

📦 배포 파일 위치
-----------------------------------------
폴더: D:\BlendBytes\AISpace\dist\AISpace-win32-x64\
실행 파일: AISpace.exe


🚀 배포 방법 3가지
-----------------------------------------

방법 1: 폴더 전체를 압축해서 배포 (가장 간단)
  1. AISpace-win32-x64 폴더를 ZIP으로 압축
  2. 사용자에게 ZIP 파일 전달
  3. 사용자는 압축 해제 후 AISpace.exe 실행

방법 2: 설치 프로그램 만들기 (전문적)
  1. NSIS 등의 인스톨러 제작 도구 사용
  2. AISpace-win32-x64 폴더 내용을 설치 프로그램으로 패키징
  3. 사용자는 설치 프로그램 실행

방법 3: 포터블 버전 (USB 메모리)
  1. AISpace-win32-x64 폴더를 USB에 복사
  2. 어디서든 AISpace.exe를 실행하면 작동


💡 테스트 방법
-----------------------------------------
1. dist\AISpace-win32-x64 폴더로 이동
2. AISpace.exe 더블클릭
3. 앱이 정상적으로 실행되는지 확인


📝 주의사항
-----------------------------------------
✓ 폴더 전체를 함께 배포해야 합니다
  (AISpace.exe 파일만 따로 떼면 안됨)

✓ Windows Defender가 경고할 수 있습니다
  (코드 서명이 없어서 발생, 정상입니다)

✓ 폴더 크기: 약 200-300MB


🎁 다음 빌드하려면
-----------------------------------------
명령어: npm run pack

dist 폴더가 자동으로 덮어씌워집니다.


📌 파일 구조
-----------------------------------------

  ├─ AISpace.exe          ← 실행 파일
  ├─ resources/           ← 앱 리소스
  ├─ locales/             ← 언어 파일
  └─ (기타 필요한 파일들)
